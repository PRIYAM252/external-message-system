External Message System

A real-time group chat application built using Node.js, Express.js, Socket.io, HTML, CSS, and JavaScript. The system detects and highlights messages from external users based on their email domains, helping improve communication security within groups.

## Features

* Real-time messaging with Socket.io
* External message detection and alerts
* Online user count tracking
* Internal and external user identification
* Simple and responsive user interface

## Technologies Used

* Node.js
* Express.js
* Socket.io
* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder:

   ```bash
   cd external-message-system
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the server:

   ```bash
   node server.js
   ```

5. Open in your browser:

   ```
   http://localhost:5000
   ```

## Project Objective

The project aims to create a secure group communication platform that can identify and notify users about messages coming from external domains, similar to enterprise messaging tools such as Microsoft Teams and Slack
